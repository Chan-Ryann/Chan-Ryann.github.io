WASSERSTEIN GAN WITH ADAPTIVE DISCRIMINATOR AUGMENTATION FOR CIFAR10 (Tensorflow/Keras)

This was my first experience working with Generative Adversarial Networks (GANs). My machine learning project involved building and training a GAN using the CIFAR10 dataset. The focus of the project was to explore an adaptive discriminator augmentation method used in state-of-the-art GAN models. Additionally, I used the Wasserstein GAN (WGAN) to stabilize the training process and improve the quality of the generated images. The aim of the project was to generate 1000 new images using the CIFAR10 dataset. Through this project, I was able to gain a deeper understanding of GANs and their potential applications in the field of machine learning.

This project gave me hands-on experience in the implementation of RL algorithms and allowed me to understand the key concepts and techniques used in solving such problems. ADVANCED DATA PREPROCESSING, ANALYTICS AND VISUALISATION ON LARGE DATASET AND SETTING UP MACHINE LEARNING PIPELINE (Python, scikit-Learn, Tableau)

This project aimed to determine if a taxi driver was considered safe or dangerous based on their driving telemetry. We worked with a large dataset containing over 7 million rows and used advanced data preprocessing methods such as feature generation, feature aggregation, outlier treatment and data balancing using techniques like SMOTE and undersampling. We then developed a machine learning pipeline to train a model that could predict if a Taxi driver is a safe or dangerous driver based on their telemetry, and used Tableau to visualize some of our findings about the relationships between various telemetry readings and safe or dangerous classification. This project provided me with hands-on experience in working with large datasets, advanced data preprocessing methods, machine learning pipeline and data visualization.

TIME SERIES USING ARIMA ON AIR POLLUTION DATASET (Python)

The goal of this project was to develop a time series model that could accurately predict the next 60 days of air pollution levels for specific gases. To achieve this, I applied various techniques for data engineering, such as identifying and removing outliers, using the AdFuller stationary test, and analyzing trends and seasonality through seasonal decomposition. Additionally, I used gridsearch to find the optimal parameters for my final model, and then used this model to make predictions for the next 60 days of gas levels. Overall, this project provided me with hands-on experience in using time series analysis, data engineering, and model optimization to make predictions about real-world data.


DATA CLUSTERING ON MALL CUSTOMERS DATASET (Python)

I selected the model that yielded the most accurate and clear results. After identifying my final model, I conducted visual analysis to understand how the data was grouped and identified 6 different demographic segments within the Mall Customers dataset. Overall, this project provided me with hands-on experience in data clustering, visual analysis, and identifying customer segments.


TECHNICAL PAPER ON CLUSTERING FOR THE WORLD HAPPINESS REPORT

This paper helps to determine what factors a country must have for its citizens to be happy by leveraging unsupervised machine learning models and data analysis on the world happiness report 2019 dataset by training and predicting and evaluating model using K-means and assigning countries to distinct clusters. The dataset consists of 156 rows and 9 columns. For each country we are given Score, GDP per capita, social support, Healthy life expectancy, Freedom to make life choices, Generosity and Perceptions of corruption. Through the use of machine learning methods, python, and different packages such as scikit learn, NumPy and pandas we attempt to cluster the countries together based on these variables.
